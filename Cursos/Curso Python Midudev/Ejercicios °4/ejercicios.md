## Pregunta 1

¿Cuál es el resultado de este código?

```python
lista2 = ["manzanas", "peras", "plátanos"]
print(lista2[1])
```

- [ ] peras
- [ ] plátanos
- [ ] Error de índice
- [ ] manzanas

## Pregunta 2

¿Qué resultado obtiene el siguiente código?

```python
lista1 = [1, 2, 3, 4, 5]
print(lista1[::2])
```

- [ ] [1, 2, 3, 4, 5]
- [ ] [2, 4]
- [ ] [1, 3, 5]
- [ ] Error de sintaxis
